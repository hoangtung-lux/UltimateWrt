# Releases

Do not commit firmware images directly into Git.

Publish heavy firmware artifacts through GitHub Releases, for example:

- `openwrt-x86-64-generic-ext4-combined-efi.img.gz`
- `openwrt-x86-64-generic-ext4-combined.img.gz`
- `sha256sums`
- `manifest`
- `config.buildinfo`
- `feeds.buildinfo`
- `version.buildinfo`
